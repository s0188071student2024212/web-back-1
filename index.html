1. С помощью команды ping получаем IP адрес домена kubsu.ru

2. Получаем А-записи и MX-записи доменов kubsu.ru, kubsu-dev.ru с помощью команды nslookup

3. Получаем дату регистрации домена kubsu.ru с помощью команды whois

4. Клонируем репозиторий с помощью команды git clone в папку www как указано в задании

4.1 Страница открывается по адресу http://ваш_логин.kubsu-dev.ru/каталог_задания_1/

5 .С помощью программы filezilla соединился с учебным сервером и скопировал на локальный компьютер файлы задания

6. Добавив новые скриншоты в репозиторий, добавляем изменения на сервер с помощью команды git pull
