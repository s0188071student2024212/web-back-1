1. С помощью команды ping получаем IP-адрес веб-сервера kubsu.ru

2. Получаем А-записи и MX-записи доменов kubsu.ru, kubsu-dev.ru с помощью команды nslookup

3. Получаем дату регистрации домена kubsu.ru с помощью команды whois

4.1 Клонируем репозиторий с помощью команды git clone в папку www

4.2 Страница открывается по адресу http://ваш_логин.kubsu-dev.ru/каталог_задания_1/

5. С помощью программы filezilla соединился с учебным сервером и скопировал на локальный компьютер файлы задания

6. После добавления новых скриншотов в репозиторий, добавляем изменения на сервер с помощью команды git pull
